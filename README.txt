*Description:
              amiq_apb VIP is an universal verification component (UVC) modeling the AMBA APB protocol

              amiq_apb package is build on top of uagt package.
              uagt is an Universal Agent, build in SystemVerilog, following UVM metodology.
              uagt sources can be downloaded from https://github.com/amiq-consulting/uagt

*Running the example:

              You need to set system variable UAGT_HOME to the location of uagt sources.
              You need to run the script ${AMIQ_APB_DIR}/demo.sh

              For more options of demo.sh you can run: ${AMIQ_APB_DIR}/demo.sh -help
