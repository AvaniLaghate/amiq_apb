*Description:
              amiq_apb VIP is an universal verification component (UVC) modeling the AMBA APB protocol

              amiq_apb package is build on top of cagt package.
              cagt is a generic agent, build in SystemVerilog, following UVM methodology.
              cagt sources can be downloaded from https://github.com/amiq-consulting/cagt

*Running the example:

              You need to set system variable CAGT_HOME to the location of cagt sources.
              You need to run the script ${AMIQ_APB_DIR}/demo.sh

              For more options of demo.sh you can run: ${AMIQ_APB_DIR}/demo.sh -help
